Aegisub_to_Anki 使用指南

从 Aegisub 字幕自动创建 Anki 卡片，支持音频、截图和视频提取。

快速开始

  1. 在 Anki 中安装 AnkiConnect 插件（工具 → 插件 → 获取插件，代码：2055492159）
  2. 启动 Anki 并保持运行
  3. 在 Aegisub 中打开视频和字幕文件
  4. 选中要制卡的字幕行
  5. 点击菜单：自动化 → 导出选中字幕行发送到Anki
  6. 在弹出的配置对话框中设置牌组和模板（首次使用需配置）

配置参数说明

基础配置

参数 说明 示例
deck_name Anki 牌组名称 Default
model_name Anki 模板名称 Basic

提取模式 (extraction_mode)

模式 说明 速度
audio_snapshot_video 提取音频 + 截图 + 视频（完整） 较慢
audio_snapshot 提取音频 + 截图 较快
audio 仅提取音频 最快

视频质量参数

参数 范围 推荐值 说明
video_crf 0-51 24 数值越小质量越高，文件越大
video_preset ultrafast ~ veryslow veryfast 越慢质量越好，编码时间越长
audio_bitrate 64k ~ 256k 128k 越高音质越好，文件越大
snapshot_quality 1-31 3 数值越小质量越高

字段映射 (field)

将 Aegisub_to_Anki 的字段映射到你的 Anki 模板字段：

Aegisub_to_Anki 字段 说明 格式
Audio 音频字段 [sound:filename.mp3]
Video 视频字段 [sound:filename.mp4]
SnapShot 截图字段 <img src="filename.jpg">
Expression 原文（\N 之前的内容） 文本
Meaning 译文（\N 之后的内容） 文本
Note 笔记字段 文本
字幕文本分割规则：
  • 有 \N："Hello\N你好" → Expression="Hello", Meaning="你好"
  • 无 \N："Hello" → Expression="Hello", Meaning=""
  • 多个 \N："Hello\N你好\N世界" → Expression="Hello", Meaning="你好<br>世界"

配置文件示例

配置文件位置：C:\Users\用户名\AppData\Roaming\Aegisub\toanki_temp\config.json

{
    "deck_name": "Default",
    "model_name": "Basic",
    "extraction_mode": "audio_snapshot_video",
    "video_crf": 24,
    "video_preset": "veryfast",
    "audio_bitrate": "128k",
    "snapshot_quality": 3,
    "show_window": false,
    "field": {
        "Audio": "Audio",
        "Video": "Video",
        "SnapShot": "Snapshot",
        "Expression": "Expression",
        "Meaning": "Meaning",
        "Note": "Notes"
    }
}

常见问题

1. 制卡失败怎么办？

  • 确认 Anki 正在运行
  • 确认 AnkiConnect 插件已安装并启用
  • 检查牌组名称和模板名称是否正确
  • 检查模板字段映射是否匹配

2. 提示"FFmpeg未安装"？

下载 FFmpeg 并添加到系统环境变量 PATH 中。

3. 卡片重复怎么办？

Aegisub_to_Anki 使用 Id 字段防止重复。如果模板有 Id 字段，相同时间段的字幕会被识别为重复。

4. 修改了 Anki 模板字段后制卡失败？

删除缓存文件：C:\Users\用户名\AppData\Roaming\Aegisub\toanki_temp\model_fields_cache.json

⚠️ 注意事项：
  • 首次使用需要在配置文件中设置正确的牌组和模板名称
  • 字段映射的右侧值必须与你的 Anki 模板字段名完全一致
  • 批量制卡时建议先测试单张卡片
  • 模板字段缓存有效期为 24 小时
  • 一定要用特定版本的FFmpeg(v6.1.2-27),不然会影响制卡速度

高级功能

批量制卡

选中多行字幕后执行，Aegisub_to_Anki 会自动批量提取媒体并创建卡片，大幅提升效率。

卡片 ID 格式

格式：视频名_时:分:秒.毫秒-时:分:秒.毫秒

示例：S01E01_0.00.01.570-0.00.03.140

调试模式

将 show_window 设为 true 可查看详细的处理过程和错误信息。
