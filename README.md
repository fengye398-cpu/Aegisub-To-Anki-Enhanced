# Aegisub To Anki Enhanced
1.这是一个通过Aegisub批量制作Anki牌组卡片的软件This is a program that creates Anki flashcard decks in batches using Aegisub.

2.要把ffmpeg三件套加入到系统变量

3.尽量用我提供的ffmpeg版本，切割速度会更快，不保证用其它版本影响切割速度

4.教程视频:https://www.bilibili.com/video/BV1jrjB62EEj/

5.参考项目:https://github.com/zhaohanim/AegisubToAnki
